# zG-Books

## Description

El proyecto **zG-Books** tiene como fin gestionar la siguiente informacion de los libros que se requieran:

- Título
- Autor
- Fecha de publicación
- Editorial
- Resumen/Sinopsis
- Género

## Funcionalidades

- Buscar/filtrar libros por:
  - Autor
  - Titulo
  - Género
  - Fecha de publicación
  - Editorial
- Agregar nuevos libros
- Eliminar libros
